Projet Gestion des équipements

Contexe

Au cours de ma première année en BTS SN, j'ai été chargé de créer un site internet permettant la gestion des équipements d'une salle de lycée, avec une durée de trois mois pour le réaliser.

Besoin

L'objectif de ce projet était de permettre aux utilisateurs présents dans un lycée d'ajouter une salle , et de pouvoir ajouter à celle-ci une prise, un type d'équipement (PC), ainsi que de pouvoir associer à chaque type d'équipement une adresse IP, une adresse MAC et un masque de sous-réseau, afin de faciliter la gestion et l'utilisation des équipements dans la salle.

Langages de Programmation & Outils

HTML 5

CSS

PHP

SQL

PHPMyAdmin

VsCode

XAMPP

Réalisation

Connexion

L'utilisateur pourra se connecter en entrant un identifiant ainsi qu'un mot de passe qui seront stockés dans une base de données. Une fois connecté, il aura accès à différentes fonctionnalités.

Ajouter une salle

Une fois connecté, l'utilisateur pourra ajouter une salle présente dans le lycée.

Ajouter une prise

Avant d'ajouter une prise, l'utilisateur doit d'abord sélectionner une salle en utilisant une liste déroulante. Ensuite, il peut nommer la prise en suivant le format suivant : A_X_YY. Cette fonctionnalité n'est disponible qu'après la connexion de l'utilisateur.

Ajouter un type d'équipement

L'utilisateur a également la possibilité d'ajouter un type d'équipement. Toutefois, cette fonctionnalité est uniquement accessible une fois que l'utilisateur s'est connecté.

Ajouter un équipement

Pour ajouter un équipement, l'utilisateur doit sélectionner une salle, une prise et un type d'équipement dans des listes déroulantes. Ensuite, il doit fournir les informations suivantes : le nom de l'équipement, l'adresse MAC (au format AA:AA:AA:AA:AA:AA), l'adresse IP (au format décimal pointé) et le masque (au format décimal pointé ou CIDR). Il est important de noter que cette fonctionnalité n'est accessible qu'après que l'utilisateur se soit connecté

Afficher les données

L'utilisateur a la possibilité de viualiser tous les éléments qu'il a ajouté à une salle.

Modifier un équipement

L'utilisateur peut modifier les éléments en sélectionnant ceux qu'il souhaite modifier et en ajoutant de nouveaux éléments. Toutefois, cette fonctionnalité n'est accessible qu'après que l'utilisateur se soit connecté.

Supprimer un équipement

L'utilisateur doit sélectionner l'équipement à supprimer dans une liste déroulante. Cette fonctionnalité n'est accessible qu'après connexion
